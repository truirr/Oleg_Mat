# Oleg_Mat Project

## Описание
Це мій проект домашнього завдання. Тут є HTML, CSS та JS файли, а також структура папок для шрифтів, зображень та додаткових сторінок.

## Функциональность
- Відображення головної сторінки `index.html`
- Базові стилі та скрипти для проекту
- Папки `fonts`, `images`, `pages` для організації ресурсів

## Особенности
- Використовується Git и GitHub Pages для публікації сайту
- Мінімум 5 коммітів для історії змін
